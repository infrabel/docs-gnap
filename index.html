The starting point for building modern web applications

GNaP is a high-productivity single page application toolset that integrates the components and APIs you need for modern web application development.

One line install using npm: npm install -g generator-gnap

Built on AngularJS, Yeoman and grunt

Nobody likes staring at a blank page. GNaP comes pre-configured with several Yeoman generators that make you more productive by writing less boilerplate code.

LIFT Application Structure

The LIFT principle provides a consistent structure that scales well, is modular, and makes it easier to increase developer efficiency by finding code quickly. Instead of organizing code by class type we prefer to group files by feature. 

src/
--- app/
------- main/
----------- todos/
--------------- list/
------------------- list.state.js
------------------- list.controller.js
------------------- list.html
--------------- new/
------------------- new.state.js
------------------- new.controller.js
------------------- new.html

AngularJS Style Guide

We care about crafting quality-first applications as much as you do. That's why we've following the AngularJS Style Guide by @john_papa: the opinionated style guide for syntax, conventions, and structuring AngularJS applications. These styles are based on development experience with AngularJS, presentations, Pluralsight training courses and working in teams.
